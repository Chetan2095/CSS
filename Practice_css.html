CSS Outline

An outline is a line that is drawn around elements, OUTSIDE the borders, to make the element "stand out" >>> outline:2px solid rgb(12, 12, 12);

The outline-offset property adds space between an outline and the border of an element. The space between border and its outline is transparent.
